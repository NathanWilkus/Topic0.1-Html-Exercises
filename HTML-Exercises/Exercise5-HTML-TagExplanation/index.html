HTML Commands:

<video>
  • Purpose - to embed a video into a document
  • Example - <video width="150" src="https://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=&cad=rja&uact=8&ved=2ahUKEwjIuYum6uDzAhUxhOAKHWbdBcEQtwJ6BAgCEAM&url=https%3A%2F%2Fwww.youtube.com%2Fwatch%3Fv%3DAFQywiW6kNQ&usg=AOvVaw2qkAFeyXIMhqtpocZ4GK-6">
  • Attributes:
    • Attribute Value Meaning
    src URL Specifies the original URL of the file
    autoplay autoplay commands the video to play as soon as it is ready
    controls control specifies that the video controls be displayed
    height pixels sets the video height
    width pixels sets the video width
    loop loop specifies the video will automatically repeat play when it is finished
    muted muted specifies that the video audio be muted
    poster URL specifies an image to be display until the video is downloaded and play is pressed


<h1> , </h1>
  • Purpose - the largest header to designate the size and color of font and background characteristics for the header which may serve as a page title
  • Example - <h1 style=background-color:red; color:black padding:15> Header </h1> indicates that the header will have a red background 15 pixels in depth with black text
  • Attributes:
    • Attribute Value Meaning
    Supports global and event attributes for text style as well as when to run text N/A N/A
